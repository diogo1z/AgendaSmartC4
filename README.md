# AgendaSmart C4

Diagrama C4 para o projeto AgendaSmart feito para o trabalho de conclusão de curso da Pós-Graduação em Arquitetura de Sistemas da PUC Minas

## Como utilizar

Siga os passos para obter o [PlantUML](https://plantuml.com/starting)

Com o PlantUML você pode gerar os diagramas a partir dos arquivos .puml ou gerar pelo vscode utilizando essa [extensão](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml) para o PlantUML
